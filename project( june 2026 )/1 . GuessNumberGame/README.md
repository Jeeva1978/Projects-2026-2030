# Number Guessing Game 🎮

A Java console game built without any reference.

## Features
- Custom number range set by user
- Higher / Lower hints
- Attempt counter
- Play again option

## Technologies
- Java
- OOP concepts
- Random + Scanner classes

## How to Run
1. Clone this repo
2. Open GuessNumberGame.java in any Java IDE
3. Run the file
4. thank you
